NeuronDigitalConfig haldls::v2::Chip::get_neuron_digital_config ( halco::hicann_dls::v2::NeuronOnDLS const &  neuron) const

Returns the digital configuration of the specified neuron.

Note
To prevent misconfiguration, enabling the external current input and external voltage output switches is only possible on the Chip level. Consequently the corresponding bits of the returned configuration container are always disabled.
