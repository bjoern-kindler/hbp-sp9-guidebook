Detailed Description

Configuration container for the two mux and buffer blocks for voltage readout.

Allows selection of various signal sources on chip and drives the signals. To reach the pads, the buffer_to_pad option in the ReadoutMuxConfig needs to be set.

Some of the selectable voltages in this container can also be selected in the ReadoutMuxConfig directly. Note that there the connection is direct, not buffered, while here the signal is amplified. Do not enable both connections, as there will be feedback otherwise!
